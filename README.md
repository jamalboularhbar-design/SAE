# Cursor Launchpad Course

A responsive, graphically rich, interactive course for learning Cursor and using
it to ship better projects.

## What is included

- A polished landing page for the Cursor learning path
- Eight beginner-friendly modules with completion tracking
- A prompt builder for practicing strong Cursor prompts
- Interactive quizzes with answer feedback and scoring
- A shipping checklist that reinforces repeatable project habits
- Responsive layouts for desktop, tablet, and mobile screens

## Easiest way to open the course

You do not need to use the terminal or type commands.

### On Windows

Double-click:

```text
Open Cursor Course.bat
```

If you want a desktop icon:

1. Right-click `Open Cursor Course.bat`.
2. Choose **Show more options** if Windows shows that menu.
3. Choose **Send to**.
4. Choose **Desktop (create shortcut)**.
5. Use that desktop shortcut whenever you want to open the course.

### On Mac

Double-click:

```text
Open Cursor Course.command
```

### On Linux

Double-click:

```text
Open Cursor Course.desktop
```

### Any computer

You can also double-click:

```text
Open Cursor Course.html
```

or:

```text
index.html
```

## Optional: run as a local website

This is only needed if you specifically want to run the course as a local
website.

```bash
npm run start
```

Then open `http://localhost:4173`, or use Cursor's port preview for port
**4173**.

## Check JavaScript

```bash
npm run check
```
